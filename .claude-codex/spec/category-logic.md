# カテゴリ判定ロジック

> 最終更新: 2026-05-07
> 役割: 操作ガイド（「カテゴリを付けて」と言われたときに読む）

---

## 基本方針

- 投稿ごとに当てはまるカテゴリをすべて付与する（複数付与前提）
- 迷ったときは付ける（漏れより多め付与を許容）
- 完全に分類できなくてよい。「たぶんこれ」で付ける

---

## カテゴリ別判定ルール

### 思想
価値観・哲学・人生観・世界の捉え方に関する考察。
「〜とはどういうことか」「〜の定義は」「〜であるべき理由」などの抽象的思考が主体の投稿。

### メンタリティー
精神的な強さ・マインドセット・心のあり方に関する話。
「メンタル」「気持ち」「感情のコントロール」「折れない」「強さ」といった文脈が中心の場合。
思想と共存することが多い。

### ストラテジー
長期的な目標・人生設計・方針の話。
具体的なアクションより「方向性・方針・長期軸」の話をしているとき。
プロジェクトと異なり、まだ実行フェーズに入っていないことが多い。

### 認知・行動
行動設計・習慣・認知のクセ・環境設計に関する話。
「〜という仕組みを作った」「〜するようにしている」「環境に置いておく」など、
日常の行動パターンを意図的に設計・変更している文脈。

### 仕事
仕事・キャリア・職場・収入に関する話題。

### プロジェクト
現在進行中の具体的な取り組み・制作・開発の話。
ストラテジーより具体的で、「今やっていること・過程」が主体。

### やりたい事
将来やりたいこと・ウィッシュリスト的な内容。
「〜したい」「いつか〜しようと思っている」が投稿の主題のとき。
やりたいことに付随する思想やメモと共存可。

### AI
AIに関する話題・考察・体験・ツール活用など。

### 技術
プログラミング・エンジニアリング・テクノロジー全般。
AIと共存することが多い。

### 言語
言語学・コミュニケーション・言葉そのものに関する考察。
言葉の使い方・語源・言語構造などが話題のとき。

### 言葉
単語・用語・概念を学んだときのメモ投稿。
「〇〇とは〜のことです」「〇〇という言葉を知った」のように、言葉の定義や発見を記録しているとき。
言語よりも「単語単位の学習記録」に近い。言語と共存可。

### 引用
**付与条件（両方満たす場合）：**
1. `>` 記法 または「○○さん曰く」「〜によると」「〜が言うには」等の明示的な引用表現がある
2. その引用が投稿の主体であり、引用に対するコメントが中心の場合

**付与しない場合：**
- 文中に `>` なしで他者の言葉が軽く登場する程度
- 引用は補足で、自分の思考が主体の場合

### ブックマーク
外部情報・知識・how-toの保存が主目的の投稿。
自分の見解より「情報そのもの」がメインのとき。
面白投稿・転載由来でhow-toや知識が含まれる場合にも付与。
メモと共存可。

### メモ
気づき・思ったこと・記憶の外部化。
「〜と気づいた」「〜と感じた」「〜だと思う」といった個人的な発見・感想が主体。
ブックマークした情報に対する所感もメモに該当する。

### 健康
身体・メンタルヘルス・食事・運動・睡眠に関する話。
食べ物・料理と共存可。

### 食べ物
食べたものの感想・グルメ・飲食の体験。
健康と共存可。

### 料理
料理を作ること・レシピ・調理方法に関する話。
食べ物と共存可。

### 体験
日常の出来事・体験記・感想。
特定カテゴリに分類されない日常のエピソード全般。

### ゲーム
ゲームに関する話題・感想・考察。

### お絵描き
絵を描くこと・イラスト・絵に関する話題。

### 人間関係
対人関係・コミュニケーション・距離感の設計に関する話。

### お役立ち
実用的な情報・tips・便利な知識。他者向けである必要はない。
メモの中でも実利・実用性があるものにはお役立ちも付与する。
「〜すると便利」「〜を知っておくといい」「〜という方法がある」的な内容。

### お金・投資
金銭・資産・投資・節約・経済に関する話題。

### 書籍
本・読書・書籍の内容に関する話題。

### マンガ・アニメ
マンガ・アニメに関する話題・感想・考察。

### 面白投稿
**自動判定不可。明示的なフラグが必要。**
ネットで見つけた面白い投稿・コンテンツの転載・共有。
投稿フォームの「転載」フラグ、またはMDフロントマターの `repost: true` が設定されている場合に付与。

転載元の内容によって共存カテゴリが変わる：
- how-to・知識系 → **ブックマーク**も付与
- それを見て思ったこと・やりたい事が出てきた → **メモ**や**やりたい事**も付与

### もちさん設定
もちさんに関するファンタジー・キャラクター的な設定の話。
「もちパワー」「やわらかい」など、現実のエピソードではなく神話的・キャラクター的に語られているとき。
現実の出来事・自分の話をしているだけでは付与しない（そちらはもち関連）。

### もち関連
もちさんが主体として登場する投稿全般。
もちさん設定と共存することが多い。
もちさんの日常・行動・エピソードなど。

## 転載フラグの設計

面白投稿カテゴリを使うときに必要な転載の明示方法。

### 投稿フォーム（UI）
「転載元」フィールドを用意し、入力があれば `面白投稿` カテゴリを自動的に候補に上げる。

### MDフロントマター（Claudeによる投稿モード）
```markdown
---
title: タイトル
account: moti
repost: true
repost_from: Twitter / Reddit / など（任意）
---
本文
```

### 投稿JSONへの追加フィールド
```json
{
  "repost": false,
  "repost_from": null
}
```

## 迷いやすいケースの対応表

| 状況 | 付与するカテゴリ |
|---|---|
| 引用が主体で自分の感想は少ない | 引用 |
| 引用もあるが自分の思考が主体 | 思想 or メモ（引用は付けない） |
| ネットの情報を保存したい | ブックマーク |
| 情報を見て気づきがあった | ブックマーク + メモ |
| メモの内容に実利・実用性がある | メモ + お役立ち |
| 転載で面白いだけ | 面白投稿 |
| 転載でhow-toも含む | 面白投稿 + ブックマーク |
| もちさんが「やわらかい」と言う | もちさん設定 + もち関連 |
| もちさんが自分の行動を語る | もち関連（設定はつけない） |
| 長期目標の話 | ストラテジー |
| 目標に向けた現在の取り組み | プロジェクト（ストラテジーも共存可） |
