# Wikipedia URL付与ロジック

> 最終更新: 2026-05-07

---

## 概要

投稿本文が特定の語・概念の説明である場合、日本語 Wikipedia の該当記事 URL を `url` フィールドに付与する。

---

## 付与するケース

以下をすべて満たす投稿に付与する：

| 条件 | 例 |
|---|---|
| 投稿タイトルまたは本文冒頭が「語の定義・説明」である | 「レトリックとは〜」「フィトケミカル〜」 |
| 日本語 Wikipedia に該当記事が存在する | `ja.wikipedia.org/wiki/レトリック` が存在する |
| 投稿者が外部リンクをすでに指定していない | `url` が null である |

---

## 付与しないケース

- 個人的な感想・日常・エピソード（もち関連・体験）が主体の投稿
- Wikipedia 記事が存在しない造語・固有名詞
- 既に `url` が設定されている投稿
- 概念の言及が補足程度で、定義説明ではない投稿

---

## URL フォーマット

```
https://ja.wikipedia.org/wiki/{パーセントエンコードされた記事名}
```

記事名は Wikipedia の実際のページタイトルに合わせる。
英語タイトルよりカタカナ・漢字のタイトルが優先される（日本語版のタイトルに従う）。

例：
- レトリック → `https://ja.wikipedia.org/wiki/%E3%83%AC%E3%83%88%E3%83%AA%E3%83%83%E3%82%AF`
- フィトケミカル → `https://ja.wikipedia.org/wiki/%E3%83%95%E3%82%A3%E3%83%88%E3%82%B1%E3%83%9F%E3%82%AB%E3%83%AB`

---

## 運用タイミング

- **カテゴリ付与と同時に確認する**（別ステップにしない）
- 付与候補がある場合は評価案・カテゴリ案と一緒に一括提示し、OKを取ってから書き込む
- 評価不要（`no_eval: true`）の投稿でも URL 付与は行う

---

## 「言葉」カテゴリとの関係

語の説明投稿は「言葉」カテゴリとセットで付与することが多い。
ただし「言葉」カテゴリ ≠ Wikipedia URL 必須ではない（慣用句・造語などは URL なしでもよい）。
